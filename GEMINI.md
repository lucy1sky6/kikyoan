# Webサイト「桔梗庵 & greengrass」制作プロジェクト

## 1. 概要

「桔梗庵」と「greengrass」という２つの施設を紹介するWebサイトを制作します。
ランディングページで両施設を紹介し、写真をクリックすると各施設ごとの詳細ページに遷移する構成となります。

## 2. 技術スタック

- **フロントエンド:** React.js
- **スタイリング:** Bootstrap
- **ルーティング:** React Router

## 3. デザイン方針

- **配色:** ランディングページは画面を縦に2分割し、左半分（桔梗庵セクション）を薄紫色、右半分（greengrassセクション）を蛍光黄緑色とします。
- 写真が主役となるような、モダンでクリーンなデザイン。
- 手書きのタイトル画像を活かし、温かみのある雰囲気も加える。
- レスポンシブ対応し、スマートフォンでも快適に閲覧できること。

## 4. ページ構成（コンポーネント案）

- **ランディングページ (`Landing.js`)**
  - `Header.js`: サイト共通のヘッダー。
  - `KikyoanSection.js`: 「桔梗庵」の紹介セクション。写真クリックで詳細ページへ。
  - `GreengrassSection.js`: 「greengrass」の紹介セクション。写真クリックで詳細ページへ。
  - `Access.js`: 共通のアクセス情報。
    - Google Mapの埋め込み
    - 交通手段（飛行機、バス、車）ごとのアクセス方法
  - `Contact.js`: 共通の連絡先情報。
  - `Footer.js`: サイト共通のフッター。

- **詳細ページ**
  - `KikyoanPage.js`: 桔梗庵の詳細情報ページ。ギャラリーの上（または下）に施設全体の紹介文を配置。ギャラリーは、インスタグラムのように8枚の正方形の写真をグリッドレイアウトで配置し、各写真の下には短い説明文を表示できる。
  - `GreengrassPage.js`: greengrassの詳細情報ページ。ギャラリーの上（または下）に施設全体の紹介文を配置。ギャラリーは、インスタグラムのように8枚の正方形の写真をグリッドレイアウトで配置し、各写真の下には短い説明文を表示できる。

## 5. アセット

- `public/images` ディレクトリに、サイトで使用する画像を配置します。
  - `kikyoan-title.png` (仮): 桔梗庵の手書きタイトル画像
  - `greengrass-title.png` (仮): greengrassの手書きタイトル画像
  - その他、各施設の写真素材

## 6. 実装手順

1. **Reactプロジェクトの初期化:**
   - `npx create-react-app kikyoan-greengrass` を実行します。
2. **ライブラリのインストール:**
   - `npm install bootstrap react-router-dom` を実行します。
3. **コンポーネントの作成:**
   - 上記構成案に基づき、各コンポーネントのファイルを作成します。
4. **ルーティングの設定:**
   - `App.js` でReact Routerを使用し、ランディングページと各詳細ページのルーティングを設定します。
5. **画像アセットの配置:**
   - `public/images` にプレースホルダー画像を配置します。
6. **動作確認:**
   - `npm start` で開発サーバーを起動し、ブラウザで表示を確認しながら開発を進めます。

以上の計画で進めます。
